## Get-OcmsSensitivityLabelPolicy

### Synopsis
Sensitivity information retrieval tool.

### Description
Gather sensitivity labeling with the option to input a user, should you want to get mailbox information for that user at the same time.

### Parameters

| Parameter       | Type     | Required | Defaults   | Description   |
|:---             |:---:     |:---:     |:---        | :---           |
| `User`          | String   | No      | None       | User to retrieve mailbox information for |


### Examples

#### Basic Usage
`Get-OcmsSensitivityLabelPolicy`

#### Getting Information Including User Mailbox
`Get-OcmsSensitivityLabelPolicy`
